# Screen Map

## Mobile

- Splash
- Onboarding carousel
  - Enter Your Sacred Space
  - Join Sunday Sessions
  - Unlock Session Recordings
- Email OTP/magic-link auth
- Bottom tabs:
  - Home
  - Sessions
  - Meditations
  - More
- Stack screens:
  - Session detail and Sacred Access Key entry
  - Audio player
  - Programs list and detail
  - Events list and detail
  - Videos
  - About Sacred Circle
  - Contact
  - My Profile
  - Help

## Admin

- Login
- Dashboard
- Sessions, including Sacred Access Keys
- Meditations
- Pages, including Programs, Events, Videos, About, Contact, and Home snippets
- Users, including manual session unlock records
- Announcements
- Settings

## Navigation Philosophy

The app keeps the main actions obvious: join Sunday session, listen, unlock recording, and contact Sacred Circle. Programs, Events, Videos, About, and Contact live inside More so the bottom navigation remains simple.
